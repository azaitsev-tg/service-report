Мобильное веб-приложение (Progressive Web App) корпоративного уровня для инженеров сервисного отдела. Предназначено для автоматизации сбора, форматирования и отправки технической отчетности с объектов (ремонт, диагностика, регламентное ТО профессионального фитнес-оборудования) напрямую в офис.

## 🎯 Основная проблематика и решение

Работа на объектах часто сопряжена с отсутствием интернета (цокольные этажи) и необходимостью фиксировать десятки фотографий (шильдики, узлы, серийные номера запчастей). Ранее это приводило к зависаниям смартфонов, ошибкам в номерах заявок и путанице в темах писем.
Данное приложение переносит всю сложную логику на сторону телефона: оно локально сжимает фото, склеивает PDF, формирует идеальную тему письма и работает **полностью в офлайне**.

## 🚀 Ключевые возможности версии 2.0

* **Accordion UI и Premium Dark Mode:** Максимально компактный интерфейс. Зоны загрузки файлов свернуты в одну строку и раскрываются только по необходимости, экономя до 60% рабочего пространства экрана. По умолчанию активирована глубокая темная тема (экономия батареи OLED-экранов на объектах) с возможностью переключения на светлую.
* **True Offline (Zero Network):** Приложение не использует внешние CSS-фреймворки и загружает нативные системные шрифты (San Francisco / Roboto). Благодаря Service Worker, загрузка и работа скриптов мгновенна даже в авиарежиме.
* **Защита памяти и умное сжатие (OOM Protection):** Локальное сжатие фотографий через Canvas API гарантированно сохраняет читаемость мелких деталей и серийных номеров. Тяжелые фотоотчеты (50+ фото) используют защиту от вылетов браузера (Lazy Processing без визуальных превью).
* **Two-Step Share Архитектура:** Разделение процессов генерации PDF и вызова системного меню отправки. Исключает обрывы по таймауту на слабых устройствах.
* **Интеллектуальные маски ввода:** Защита от опечаток. Автоматическая подстановка префиксов (`P.`, `ПРОЗ-`) и новой маски `ITSM XXXXXX` (с автоисправлением ошибочной русской раскладки).
* **Раздельная логика нейминга:** * *Тема письма* копируется по одному тапу и содержит тег "+ лист заказа".
  * *Имя PDF-файла* автоматически очищается от спецсимволов и кавычек, строго сохраняя технический формат.
* **Центр онбординга (Side Drawer):** Встроенное боковое меню содержит подробное руководство пользователя, ссылки на базы знаний (каталоги запчастей, E-Learning) и кнопку авто-генерации баг-репорта, которая сама собирает данные об ОС и разрешении экрана для техподдержки.

## 🛠 Технический стек

* **HTML5 / CSS3** — чистый, автономный код без тяжелых библиотек.
* **Vanilla JavaScript (ES6+)** — предиктивный ввод, асинхронные очереди, Canvas API.
* **[pdf-lib](https://pdf-lib.js.org)** — клиентская сборка и склейка многостраничных PDF.
* **Web Share API** — нативная интеграция с системными мессенджерами и почтой.
* **Service Worker (`sw.js`) & LocalStorage** — кэширование приложения и сохранение пользовательских настроек (тема).

## 📱 Установка на устройство

Приложение работает в обход классических магазинов (AppStore / Google Play).
1. Откройте ссылку на инструмент в браузере Safari (iOS) или Chrome (Android) при **включенном интернете** (для первичного кэширования).
2. Вызовите системное меню браузера («Поделиться» / «Три точки»).
3. Нажмите **«На экран домой» (Add to Home Screen)**.
4. Приложение появится на рабочем столе и будет доступно в любых условиях.
