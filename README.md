# ⚙️ Service Report App (PWA)

[![PWA](httpsimg.shields.iobadgePWA-Ready-success.svg)](#)
[![JS](httpsimg.shields.iobadgeVanilla-JavaScript-yellow.svg)](#)
[![PDF](httpsimg.shields.iobadgepdf--lib-v1.17.1-blue.svg)](#)

Мобильное веб-приложение (Progressive Web App) для сервисного отдела. Предназначено для автоматизации сбора, форматирования и отправки технической отчетности с объектов (ремонт, диагностика, ТО профессионального фитнес-оборудования) напрямую в офис.

## 🎯 Основная проблематика и решение

Ранее инженерам приходилось вручную прикреплять десятки тяжелых фотографий, следить за форматом темы письма и бороться с отсутствием связи на объектах. 
Данное приложение переносит всю логику обработки на сторону клиента (браузер смартфона), формируя стандартизированный PDF-документ и подготавливая письмо для отправки.

## 🚀 Ключевые возможности

 Генерация PDF «на лету» Бесшовная склейка актов (PDFФото), накладных и фотографий шильдиковузлов в единый многостраничный документ.
 Локальная оптимизация изображений Автоматическое сжатие фотографий через Canvas API перед сборкой в PDF. Снижает вес итогового отчета в 10+ раз при сохранении читаемости серийных номеров.
 Предиктивный ввод масок Защита от опечаток в номерах заявок. Автоматическая подстановка префиксов (`P.`  `ПРОЗ-`) и нулей в зависимости от вводимого формата.
 Smart Offline Mode Если на объекте (в цоколе) нет интернета, приложение не блокирует работу, а предлагает сохранить сгенерированный PDF в память устройства для последующей ручной отправки.
 Стандартизация нейминга Строгое формирование темы письма и очищенного имени файла (без спецсимволов и юридических форм) для удобного поиска в базе данных и 1С.

## 🛠 Технический стек

Приложение разработано без использования тяжелых фреймворков для обеспечения максимальной скорости загрузки и совместимости
 HTML5  CSS3 — адаптивный UI, стилизованный под строгие корпоративные стандарты.
 Vanilla JavaScript — логика интерфейса, предиктивный ввод, работа с Canvas.
 [pdf-lib](httpspdf-lib.js.org) — ядро сборки PDF-документа.
 Web Share API — системная передача файлов в почтовые клиенты (с Fallback-сценарием `mailto`).
 Service Worker  Manifest — обеспечение PWA-функционала (установка на рабочий стол).

## 📱 Установка для инженеров

Приложение не требует скачивания из магазинов (AppStore  Google Play).
1. Открыть ссылку на проект в мобильном браузере.
2. Открыть системное меню браузера («Поделиться» в Safari  «Три точки» в Chrome).
3. Выбрать пункт «На экран домой» (Add to Home Screen).
