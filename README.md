# ⚙️ Service Report App (PWA)

![PWA](https://flat.badgen.net/badge/PWA/Ready/green)
![JS](https://flat.badgen.net/badge/JS/Vanilla/yellow)
![PDF](https://flat.badgen.net/badge/pdf-lib/v1.17.1/blue)

Мобильное веб-приложение (Progressive Web App) для сервисного отдела. Предназначено для автоматизации сбора, форматирования и отправки технической отчетности с объектов (ремонт, диагностика, ТО профессионального фитнес-оборудования) напрямую в офис.

## 🎯 Основная проблематика и решение

Ранее инженерам приходилось вручную прикреплять десятки тяжелых фотографий, следить за форматом темы письма и бороться с отсутствием связи на объектах. 
Данное приложение переносит всю логику обработки на сторону клиента (браузер смартфона), формируя стандартизированный PDF-документ и подготавливая письмо для отправки.

## 🚀 Ключевые возможности

* **Генерация PDF «на лету»:** Бесшовная склейка актов (PDF/Фото), накладных и фотографий шильдиков/узлов в единый многостраничный документ. Каждая фотография автоматически подгоняется под размер индивидуальной страницы без белых полей.
* **Умная корзина (Smart Basket):** Накопительная система выбора файлов. Инженер может догружать фотографии частями (не стирая предыдущие), точечно удалять ошибочные кадры или полностью очищать разделы.
* **Защита оперативной памяти (OOM Protection):** Для тяжелого фотоотчета (50+ фото) используется интерфейс «умного счетчика» без визуального предпросмотра (Lazy Processing). 
* **Локальная оптимизация изображений:** Автоматическое сжатие фотографий (Canvas API) перед сборкой в PDF. Сжатие происходит в строгой последовательной очереди (On-the-fly) через легковесные ссылки `URL.createObjectURL`, что полностью исключает вылеты вкладки браузера (особенно в iOS Safari).
* **Предиктивный ввод масок:** Защита от опечаток в номерах заявок. Автоматическая подстановка префиксов (`P.` / `ПРОЗ-`) и нулей в зависимости от вводимого формата.
* **Smart Offline Mode:** Если на объекте (в цоколе) нет интернета, приложение не блокирует работу, а предлагает сохранить сгенерированный PDF в память устройства (с четким разделением логики кнопок "Сохранить" и "Отправить").
* **Стандартизация нейминга:** Строгое формирование темы письма и очищенного имени файла (без спецсимволов и юридических форм) для удобного поиска в базе данных и 1С.

## 🛠 Технический стек

Приложение разработано без использования тяжелых фреймворков для обеспечения максимальной скорости загрузки и совместимости.

* **HTML5 / CSS3** — адаптивный UI, стилизованный под строгие корпоративные стандарты.
* **Vanilla JavaScript (ES6+)** — логика интерфейса, предиктивный ввод, работа с Canvas и асинхронными очередями памяти.
* **[pdf-lib](https://pdf-lib.js.org)** — ядро сборки PDF-документа.
* **Web Share API** — системная передача файлов в мессенджеры и почтовые клиенты (с Fallback-сценарием `mailto`).
* **Service Worker & Manifest** — обеспечение PWA-функционала (установка на рабочий стол).

## 📱 Установка для инженеров

Приложение не требует скачивания из магазинов (AppStore / Google Play).
1. Открыть ссылку на проект в мобильном браузере.
2. Открыть системное меню браузера («Поделиться» в Safari / «Три точки» в Chrome).
3. Выбрать пункт «На экран домой» (Add to Home Screen).
